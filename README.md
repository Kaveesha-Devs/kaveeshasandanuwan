# Kaveesha Muthugala – Portfolio Website

A stunning, professional portfolio built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```
Visit `http://localhost:3000`

### 3. Build for production
```bash
npm run build
```

---

## 🌐 Deploy to Vercel (Recommended – Free!)

1. Push this folder to a **GitHub repository**
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Click **Deploy** – done! You'll get a `.vercel.app` URL

Or use Vercel CLI:
```bash
npm install -g vercel
vercel
```

## ✏️ Customization

### Update your personal info:
| File | What to edit |
|------|-------------|
| `src/components/Hero.tsx` | Name, title, summary |
| `src/components/About.tsx` | Bio, soft skills, info cards |
| `src/components/Skills.tsx` | Skill categories + levels |
| `src/components/Projects.tsx` | Project cards + links |
| `src/components/Education.tsx` | Education timeline |
| `src/components/Contact.tsx` | Email, GitHub, referees |
| `src/app/layout.tsx` | Page title/meta description |

### Update your email:
Search for `kaveesha@example.com` across the project and replace with your real email.

## ✨ Features

- **Smooth animations** with Framer Motion (scroll-triggered, staggered reveals)
- **Typewriter effect** for roles in the Hero section
- **Skill bars** with animated fill on scroll
- **Project filter** (All / Web App / Mobile App / AI/ML)
- **Timeline** education section
- **Glassmorphism** UI cards
- **Custom scrollbar** and noise texture
- **Fully responsive** mobile/tablet/desktop
- **Static export ready** for any hosting
